## Olvídese del inestable historial de Windows
**ClipHistory** está diseñado específicamente para reemplazar el sistema inconsistente de Windows con un motor ligero de alto rendimiento que funciona perfectamente con lectores de pantalla.

<br />

## Teclas Rápidas
`<kbd>Windows+V</kbd>`
* **Centro de Mando:** Una sola pulsación lanza el historial al instante.
* **Limpieza Instantánea:** Doble toque para borrar todo, excepto lo fijado.
