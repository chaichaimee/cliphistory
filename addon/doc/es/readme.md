# ClipHistory
### *La experiencia definitiva del portapapeles para NVDA: estabilidad sólida, latencia cero e historial persistente que Windows por sí solo no puede ofrecer.*

<br />

author: chai chaimee
url: [https://github.com/chaichaimee/ClipHistory](https://github.com/chaichaimee/ClipHistory)

<br />

## Descripción
ClipHistory es una solución de gestión de portapapeles de alto rendimiento diseñada exclusivamente para usuarios de NVDA. Mientras que el historial estándar de Windows puede volverse lento o no responder cuando hay un lector de pantalla activo, ClipHistory proporciona una experiencia fluida y sin retrasos. Captura automáticamente cada fragmento de texto y HTML que copies, almacenando hasta 500 elementos en una lista persistente. Ya seas escritor, programador o usuario diario de la web, este complemento garantiza que tus datos copiados estén siempre organizados, editables y listos para ser reutilizados al instante sin perder el enfoque.

<br />

## Novedades
Elementos fijados mejorados: Ahora admite "Nombres para mostrar" personalizados para textos largos (más de 50 palabras) y muestra el recuento total de caracteres dentro de la lista del historial para mayor claridad.

<br />

## Teclas rápidas
<kbd>Windows+V</kbd>
*   **Pulsación única:** Abre el diálogo de ClipHistory para ver y gestionar tus fragmentos guardados.
*   **Pulsación doble (rápida):** Borra instantáneamente todos los elementos no fijados de tu historial para mantener la lista limpia.

<br />

## Características

**1. Gestión fluida del historial (Límite de 500 elementos)**  
ClipHistory funciona silenciosamente en segundo plano. Cada vez que presionas <kbd>Control+C</kbd>, el texto se guarda.  
<br />
• **Cómo funciona:** El sistema almacena hasta 500 elementos únicos. Cuando llegas al elemento 501, la entrada más antigua no fijada se elimina automáticamente para dejar espacio a la nueva. Esto asegura que tu historial se mantenga actualizado sin sobrecargar el sistema.

<br />

**2. Pegado automático mágico**  
Elimina la necesidad de pegar manualmente.  
<br />
• **Paso 1:** Presiona <kbd>Windows+V</kbd> para abrir la lista.  
• **Paso 2:** Usa las teclas de flecha para seleccionar el fragmento que necesitas.  
• **Paso 3:** Presiona <kbd>Enter</kbd>. ClipHistory cerrará automáticamente el diálogo y pegará el texto directamente en tu aplicación activa.

<br />

**3. Fijación avanzada con nombres para mostrar**  
Para contenido extenso como artículos o bloques de código, gestionar una lista puede ser difícil. ClipHistory introduce una forma inteligente de manejar fragmentos "pesados".  
<br />
• **El activador de 50 palabras:** Cuando fijas un elemento que contiene más de 50 palabras, el diálogo de edición proporcionará un campo de "Nombre para mostrar".  
• **Simplificando la lista:** Al introducir un alias corto en este campo, la lista principal solo mostrará ese nombre en lugar del bloque masivo de texto.  
• **Conocimiento de caracteres:** Incluso al usar un Nombre para mostrar, ClipHistory mostrará el recuento total de caracteres al final del elemento en la lista, para que siempre sepas el tamaño exacto de los datos que vas a pegar.

<br />

**4. Edición sobre la marcha**  
Los errores ocurren, pero puedes corregirlos antes de pegarlos.  
<br />
• **Paso 1:** Localiza el elemento en la lista de ClipHistory.  
• **Paso 2:** Elige la opción Editar. Puedes modificar el texto, actualizar el nombre para mostrar o cambiar el estado de "Fijado".  
• **Paso 3:** Guarda los cambios. El texto actualizado ya está listo para su uso, perfectamente corregido.

<br />

**5. Soporte estructural HTML**  
A diferencia de los gestores básicos, ClipHistory captura el formato del contenido web. Si copias un enlace o texto formateado de un sitio web, conserva esa estructura, convirtiéndolo en la herramienta perfecta para usuarios que trabajan en diferentes plataformas digitales.
