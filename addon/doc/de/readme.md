## Vergessen Sie den instabilen Windows-Zwischenablageverlauf
Sind Sie es leid, dass die integrierte Windows-Cloud-Zwischenablage verzögert, wenn NVDA aktiv ist? **ClipHistory** ersetzt das inkonsistente Windows-System durch eine leistungsstarke Engine, die perfekt mit Screenreadern funktioniert.

<br />

## Tastenkombinationen
`<kbd>Windows+V</kbd>`
* **Das Kommandozentrum:** Einmal drücken öffnet den Verlauf.
* **Sofortige Reinigung:** Doppeltippen löscht alles außer angepinnten Elementen.
