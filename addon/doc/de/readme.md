# ClipHistory
### *Das ultimative Clipboard-Erlebnis für NVDA: Felsenfeste Stabilität, null Latenz und ein dauerhafter Verlauf, den Windows allein nicht bieten kann.*

<br />

author: chai chaimee
url: [https://github.com/chaichaimee/ClipHistory](https://github.com/chaichaimee/ClipHistory)

<br />

## Beschreibung
ClipHistory ist eine hochperformante Lösung zur Verwaltung der Zwischenablage, die exklusiv für NVDA-Benutzer entwickelt wurde. Während der Standard-Verlauf von Windows bei aktivem Screenreader oft langsam oder instabil reagiert, bietet ClipHistory eine nahtlose, verzögerungsfreie Erfahrung. Es erfasst automatisch jeden Text- und HTML-Schnipsel, den Sie kopieren, und speichert bis zu 500 Einträge in einer dauerhaften Liste. Egal, ob Sie Autor, Programmierer oder täglicher Web-Nutzer sind – dieses Add-on stellt sicher, dass Ihre kopierten Daten stets organisiert, bearbeitbar und sofort wiederverwendbar sind, ohne jemals den Fokus zu verlieren.

<br />

## Was ist neu
Verbesserte angepinnte Elemente: Unterstützt jetzt benutzerdefinierte „Anzeigenamen“ für lange Texte (über 50 Wörter) und zeigt die Gesamtzahl der Zeichen innerhalb der Verlaufsliste für bessere Übersichtlichkeit an.

<br />

## Tastenkombinationen
<kbd>Windows+V</kbd>
*   **Einmaliges Drücken:** Öffnet den ClipHistory-Dialog, um Ihre gespeicherten Schnipsel anzuzeigen und zu verwalten.
*   **Zweimaliges Drücken (schnell):** Löscht sofort alle nicht angepinnten Elemente aus Ihrem Verlauf, um die Liste sauber zu halten.

<br />

## Funktionen

**1. Nahtlose Verlaufsverwaltung (Limit 500 Elemente)**  
ClipHistory arbeitet lautlos im Hintergrund. Jedes Mal, wenn Sie <kbd>Strg+C</kbd> drücken, wird der Text gespeichert.  
<br />
• **So funktioniert es:** Das System speichert bis zu 500 eindeutige Elemente. Wenn Sie das 501. Element erreichen, wird der älteste nicht angepinnte Eintrag automatisch entfernt, um Platz für das neue Element zu schaffen. Dies stellt sicher, dass Ihr Verlauf aktuell bleibt, ohne Ihr System zu überlasten.

<br />

**2. Magisches Auto-Einfügen**  
Machen Sie manuelles Einfügen überflüssig.  
<br />
• **Schritt 1:** Drücken Sie <kbd>Windows+V</kbd>, um die Liste zu öffnen.  
• **Schritt 2:** Verwenden Sie die Pfeiltasten, um den benötigten Schnipsel auszuwählen.  
• **Schritt 3:** Drücken Sie <kbd>Enter</kbd>. ClipHistory schließt den Dialog automatisch und fügt den Text direkt in Ihre aktive Anwendung ein.

<br />

**3. Erweitertes Anpinnen mit Anzeigenamen**  
Bei langen Inhalten wie Artikeln oder Codeblöcken kann die Verwaltung einer Liste schwierig sein. ClipHistory führt eine intelligente Methode für „schwere“ Schnipsel ein.  
<br />
• **Der 50-Wörter-Trigger:** Wenn Sie ein Element anpinnen, das mehr als 50 Wörter enthält, bietet der Bearbeitungsdialog ein Feld „Anzeigename“ an.  
• **Vereinfachung der Liste:** Durch die Eingabe eines kurzen Namens in dieses Feld zeigt die Hauptliste nur diesen Namen anstelle des massiven Textblocks an.  
• **Zeichen-Bewusstsein:** Auch bei Verwendung eines Anzeigenamens zeigt ClipHistory die Gesamtzahl der Zeichen am Ende des Elements in der Liste an, sodass Sie immer die genaue Größe der Daten kennen, die Sie einfügen möchten.

<br />

**4. Bearbeitung im Flug**  
Fehler passieren, aber Sie können sie beheben, bevor sie eingefügt werden.  
<br />
• **Schritt 1:** Suchen Sie das Element in der ClipHistory-Liste.  
• **Schritt 2:** Wählen Sie die Option Bearbeiten. Sie können den Text ändern, den Anzeigenamen aktualisieren oder den Status „Angepinnt“ umschalten.  
• **Schritt 3:** Speichern Sie die Änderungen. Der aktualisierte Text ist nun einsatzbereit und perfekt korrigiert.

<br />

**5. Strukturelle HTML-Unterstützung**  
Im Gegensatz zu einfachen Managern erfasst ClipHistory die Formatierung von Webinhalten. Wenn Sie einen Link oder formatierten Text von einer Website kopieren, bleibt diese Struktur erhalten, was es zum perfekten Werkzeug für Benutzer macht, die plattformübergreifend digital arbeiten.
