# ClipHistory
### *Максимальное удобство работы с буфером обмена для NVDA: железная стабильность, нулевая задержка и постоянная история, которую Windows сама по себе обеспечить не может.*

## Описание
ClipHistory — это высокопроизводительное решение для управления буфером обмена, разработанное эксклюзивно для пользователей NVDA. В то время как стандартная история буфера обмена Windows может работать медленно или не отвечать при активном экранном дикторе, ClipHistory обеспечивает плавную работу без задержек. Оно автоматически захватывает каждый фрагмент текста и HTML, который вы копируете, сохраняя до 500 элементов в постоянном списке. Будь вы писателем, программистом или обычным пользователем интернета, это дополнение гарантирует, что ваши скопированные данные всегда будут организованы, доступны для редактирования и готовы к мгновенному использованию без потери фокуса.

<br />

## Что нового
Улучшенные закрепленные элементы: теперь поддерживаются пользовательские «Отображаемые имена» для длинных текстов (более 50 слов) и отображается общее количество символов в списке истории для большей ясности.

<br />

## Горячие клавиши
<kbd>Windows+V</kbd>
*   **Одиночное нажатие:** открывает диалоговое окно ClipHistory для просмотра и управления сохраненными фрагментами.
*   **Двойное нажатие (быстрое):** мгновенно удаляет все незакрепленные элементы из вашей истории, чтобы поддерживать список в чистоте.

<br />

## Функции

**1. Плавное управление историей (лимит 500 элементов)**  
ClipHistory бесшумно работает в фоновом режиме. Каждый раз, когда вы нажимаете <kbd>Control+C</kbd>, текст сохраняется.  
<br />
• **Как это работает:** Система хранит до 500 уникальных элементов. Когда вы копируете 501-й элемент, самая старая незакрепленная запись автоматически удаляется, чтобы освободить место для новой. Это гарантирует актуальность вашей истории без перегрузки системы.

<br />

**2. Магическая авто-вставка**  
Забудьте о необходимости ручной вставки.  
<br />
• **Шаг 1:** Нажмите <kbd>Windows+V</kbd>, чтобы открыть список.  
• **Шаг 2:** Используйте клавиши со стрелками, чтобы выбрать нужный фрагмент.  
• **Шаг 3:** Нажмите <kbd>Enter</kbd>. ClipHistory автоматически закроет диалог и вставит текст прямо в ваше активное приложение.

<br />

**3. Продвинутое закрепление с отображаемыми именами**  
Для объемного контента, такого как статьи или блоки кода, управление списком может быть затруднительным. ClipHistory представляет интеллектуальный способ обработки «тяжелых» фрагментов.  
<br />
• **Триггер на 50 слов:** Когда вы закрепляете элемент, содержащий более 50 слов, в диалоге редактирования появится поле «Отображаемое имя».  
• **Упрощение списка:** Введя короткий псевдоним в это поле, основной список истории будет показывать только это имя вместо огромного блока текста.  
• **Учет символов:** Даже при использовании отображаемого имени ClipHistory покажет общее количество символов в конце элемента в списке, чтобы вы всегда знали точный объем данных, которые собираетесь вставить.

<br />

**4. Редактирование «на лету»**  
Ошибки случаются, но вы можете исправить их перед вставкой.  
<br />
• **Шаг 1:** Найдите элемент в списке ClipHistory.  
• **Шаг 2:** Выберите опцию «Редактировать». Вы можете изменить текст, обновить отображаемое имя или переключить статус «Закреплено».  
• **Шаг 3:** Сохраните изменения. Обновленный текст готов к использованию, идеально исправленный.

<br />

**5. Поддержка структуры HTML**  
В отличие от простых менеджеров, ClipHistory сохраняет форматирование веб-контента. Если вы копируете ссылку или отформатированный текст с веб-сайта, он сохраняет эту структуру, что делает его идеальным инструментом для пользователей, работающих на разных цифровых платформах.
