## Забудьте о нестабильной истории Windows
**ClipHistory** разработан специально для замены непоследовательной системы Windows легким и высокопроизводительным движком для экранных чтецов.

<br />

## Горячие клавиши
`<kbd>Windows+V</kbd>`
* **Командный центр:** Одиночное нажатие мгновенно открывает историю.
* **Мгновенная очистка:** Двойное нажатие удаляет всё, кроме закрепленного.
