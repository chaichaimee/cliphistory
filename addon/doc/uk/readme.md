## Забудьте про нестабільну історію Windows
**ClipHistory** розроблений спеціально для заміни непослідовної системи Windows легким та високопродуктивним движком, який бездоганно працює з екранними читцями.

<br />

## Потужні гарячі клавіші
`<kbd>Windows+V</kbd>`
* **Командний центр:** Одне натискання миттєво відкриває всю вашу історію.
* **Миттєве очищення:** Подвійне натискання видаляє все, крім закріплених елементів.

<br />

## Ключові особливості
* **Магічна авто-вставка:** Виберіть елемент, натисніть `<kbd>Enter</kbd>`, і текст буде вставлено автоматично.
* **Розумне закріплення:** Закріплені елементи захищені від очищення.
