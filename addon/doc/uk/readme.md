# ClipHistory
### *Максимальна зручність роботи з буфером обміну для NVDA: залізна стабільність, нульова затримка та постійна історія, яку Windows сама по собі не може забезпечити.*

<br />

автор: chai chaimee
url: [https://github.com/chaichaimee/ClipHistory](https://github.com/chaichaimee/ClipHistory)

<br />

## Опис
ClipHistory — це високопродуктивне рішення для керування буфером обміну, розроблене виключно для користувачів NVDA. У той час як стандартна історія буфера обміну Windows може працювати повільно або не відповідати при активному екранному читці, ClipHistory забезпечує плавну роботу без затримок. Воно автоматично захоплює кожен фрагмент тексту та HTML, який ви копіюєте, зберігаючи до 500 елементів у постійному списку. Незалежно від того, чи ви письменник, програміст або звичайний користувач інтернету, це доповнення гарантує, що ваші скопійовані дані завжди будуть організовані, доступні для редагування та готові до миттєвого використання без втрати фокусу.

<br />

## Що нового
Покращені закріплені елементи: тепер підтримуються користувацькі «Відображувані імена» для довгих текстів (понад 50 слів) і відображається загальна кількість символів у списку історії для більшої чіткості.

<br />

## Гарячі клавіші
<kbd>Windows+V</kbd>
*   **Одне натискання:** відкриває діалогове вікно ClipHistory для перегляду та керування збереженими фрагментами.
*   **Подвійне натискання (швидке):** миттєво видаляє всі незакріплені елементи з вашої історії, щоб підтримувати список у чистоті.

<br />

## Функції

**1. Плавне керування історією (ліміт 500 елементів)**  
ClipHistory безшумно працює у фоновому режимі. Щоразу, коли ви натискаєте <kbd>Control+C</kbd>, текст зберігається.  
<br />
• **Як це працює:** Система зберігає до 500 унікальних елементів. Коли ви копіюєте 501-й елемент, найстаріший незакріплений запис автоматично видаляється, щоб звільнити місце для нового. Це гарантує актуальність вашої історії без перевантаження системи.

<br />

**2. Магічна авто-вставка**  
Забудьте про необхідність ручного вставлення.  
<br />
• **Крок 1:** Натисніть <kbd>Windows+V</kbd>, щоб відкрити список.  
• **Крок 2:** Використовуйте клавіші зі стрілками, щоб вибрати потрібний фрагмент.  
• **Крок 3:** Натисніть <kbd>Enter</kbd>. ClipHistory автоматично закриє діалог і вставить текст безпосередньо у вашу активну програму.

<br />

**3. Просунуте закріплення з відображуваними іменами**  
Для об'ємного контенту, такого як статті або блоки коду, керування списком може бути складним. ClipHistory представляє інтелектуальний спосіб обробки «важких» фрагментів.  
<br />
• **Тригер на 50 слів:** Коли ви закріплюєте елемент, що містить понад 50 слів, у діалозі редагування з'явиться поле «Відображуване ім'я».  
• **Спрощення списку:** Ввівши короткий псевдонім у це поле, основний список історії показуватиме лише це ім'я замість величезного блоку тексту.  
• **Облік символів:** Навіть при використанні відображуваного імені ClipHistory покаже загальну кількість символів у кінці елемента в списку, щоб ви завжди знали точний обсяг даних, які збираєтеся вставити.

<br />

**4. Редагування «на льоту»**  
Помилки трапляються, але ви можете виправити їх перед вставленням.  
<br />
• **Крок 1:** Знайдіть елемент у списку ClipHistory.  
• **Крок 2:** Виберіть опцію «Редагувати». Ви можете змінити текст, оновити відображуване ім'я або переключити статус «Закріплено».  
• **Крок 3:** Збережіть зміни. Оновлений текст готовий до використання, ідеально виправлений.

<br />

**5. Підтримка структури HTML**  
На відміну від простих менеджерів, ClipHistory зберігає форматування веб-контенту. Якщо ви копіюєте посилання або відформатований текст із веб-сайту, він зберігає цю структуру, що робить його ідеальним інструментом для користувачів, які працюють на різних цифрових платформах.
