# ClipHistory
### *L'expérience ultime du presse-papiers pour NVDA : Stabilité sans faille, latence zéro et historique persistant que Windows seul ne peut offrir.*

<br />

author: chai chaimee
url: [https://github.com/chaichaimee/ClipHistory](https://github.com/chaichaimee/ClipHistory)

<br />

## Description
ClipHistory est une solution de gestion de presse-papiers haute performance conçue exclusivement pour les utilisateurs de NVDA. Alors que l'historique standard du presse-papiers de Windows peut être lent ou réactif lorsqu'un lecteur d'écran est actif, ClipHistory offre une expérience fluide et sans décalage. Il capture automatiquement chaque fragment de texte et HTML que vous copiez, stockant jusqu'à 500 éléments dans une liste persistante. Que vous soyez écrivain, codeur ou utilisateur quotidien du web, cette extension garantit que vos données copiées sont toujours organisées, modifiables et prêtes à être réutilisées instantanément sans jamais perdre le focus.

<br />

## Quoi de neuf
Éléments épinglés améliorés : Supporte désormais les "Noms d'affichage" personnalisés pour les textes longs (plus de 50 mots) et affiche le nombre total de caractères dans la liste d'historique pour une meilleure clarté.

<br />

## Touches de raccourci
<kbd>Windows+V</kbd>
*   **Appui simple :** Ouvre la boîte de dialogue ClipHistory pour afficher et gérer vos extraits enregistrés.
*   **Appui double (rapide) :** Efface instantanément tous les éléments non épinglés de votre historique pour garder votre liste propre.

<br />

## Fonctionnalités

**1. Gestion transparente de l'historique (Limite de 500 éléments)**  
ClipHistory travaille silencieusement en arrière-plan. Chaque fois que vous appuyez sur <kbd>Control+C</kbd>, le texte est sauvegardé.  
<br />
• **Comment ça marche :** Le système stocke jusqu'à 500 éléments uniques. Lorsque vous atteignez le 501ème élément, l'entrée la plus ancienne non épinglée est automatiquement supprimée pour faire de la place à la nouvelle. Cela garantit que votre historique reste à jour sans surcharger votre système.

<br />

**2. Auto-collage magique**  
Éliminez le besoin de coller manuellement.  
<br />
• **Étape 1 :** Appuyez sur <kbd>Windows+V</kbd> pour ouvrir la liste.  
• **Étape 2 :** Utilisez vos touches fléchées pour sélectionner l'extrait dont vous avez besoin.  
• **Étape 3 :** Appuyez sur <kbd>Entrée</kbd>. ClipHistory fermera automatiquement la boîte de dialogue et collera le texte directement dans votre application active.

<br />

**3. Épinglage avancé avec noms d'affichage**  
Pour les contenus longs comme les articles ou les blocs de code, la gestion d'une liste peut être difficile. ClipHistory introduit une manière intelligente de gérer les extraits "lourds".  
<br />
• **Le déclencheur de 50 mots :** Lorsque vous épinglez un élément contenant plus de 50 mots, la boîte de dialogue d'édition proposera un champ "Nom d'affichage".  
• **Simplification de la liste :** En saisissant un court pseudonyme dans ce champ, la liste d'historique principale n'affichera que ce nom au lieu du bloc de texte massif.  
• **Conscience des caractères :** Même lors de l'utilisation d'un nom d'affichage, ClipHistory affichera le nombre total de caractères à la fin de l'élément dans la liste, afin que vous connaissiez toujours la taille exacte des données que vous allez coller.

<br />

**4. Édition à la volée**  
Les erreurs arrivent, mais vous pouvez les corriger avant qu'elles ne soient collées.  
<br />
• **Étape 1 :** Localisez l'élément dans la liste ClipHistory.  
• **Étape 2 :** Choisissez l'option Édition. Vous pouvez modifier le texte, mettre à jour le nom d'affichage ou basculer le statut "Épinglé".  
• **Étape 3 :** Enregistrez les modifications. Le texte mis à jour est maintenant prêt à l'emploi, parfaitement corrigé.

<br />

**5. Support structurel HTML**  
Contrairement aux gestionnaires de base, ClipHistory capture le formatage du contenu web. Si vous copiez un lien ou du texte formaté à partir d'un site web, il conserve cette structure, ce qui en fait l'outil parfait pour les utilisateurs travaillant sur différentes plateformes numériques.
