## Oubliez l'historique instable de Windows
**ClipHistory** remplace le système incohérent de Windows par un moteur léger et performant qui fonctionne parfaitement avec les lecteurs d'écran.

<br />

## Raccourcis Claviers
`<kbd>Windows+V</kbd>`
* **Le Centre de Commande :** Un seul appui affiche instantanément votre historique.
* **Nettoyage Instantané :** Appuyez deux fois pour tout effacer sauf les éléments épinglés.
