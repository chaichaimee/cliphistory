## Forget the Unstable Windows Clipboard History
Are you tired of the built-in Windows Cloud Clipboard failing to respond or lagging when NVDA is active? **ClipHistory** is engineered specifically to replace the inconsistent Windows clipboard system with a lightweight, high-performance engine that works flawlessly with screen readers. No more missed captures, no more focus loss—just a professional-grade history manager that "just works" every single time.

<br />

## Powerful Hot Keys
`<kbd>Windows+V</kbd>`
* **The Command Center:** A single press launches the sleek ClipHistory Dialog, putting your entire history in view instantly.
* **The Instant Cleanup:** Double-tap to vanish all unwanted clutter. It clears your history in a flash, but smartly preserves your pinned items.

<br />

## Game-Changing Features
* **Magic Auto-Paste:** Experience true workflow fluidity. Select an item, hit `<kbd>Enter</kbd>`, and watch as ClipHistory automatically injects the text into your active field and closes itself.
* **Smart Pinning:** Secure your most vital snippets. Pinned items are shielded from any "Clear All" commands.
* **Native HTML Support:** We don't just capture text; we capture **structure**. Perfect for web developers and power users.
* **On-the-Fly Editing:** Fix mistakes before you paste. Edit any history item directly within the dialog for absolute precision.
