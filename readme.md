# ClipHistory
### *The ultimate clipboard experience for NVDA: Rock-solid stability, zero latency, and persistent history that Windows alone can't deliver.*

<br />

author: chai chaimee
url: [https://github.com/chaichaimee/ClipHistory](https://github.com/chaichaimee/ClipHistory)

<br />

## Description
ClipHistory is a high-performance clipboard management solution designed exclusively for NVDA users. While the standard Windows clipboard history can be slow or unresponsive when a screen reader is active, ClipHistory provides a seamless, lag-free experience. It automatically captures every text and HTML snippet you copy, storing up to 500 items in a persistent list. Whether you are a writer, a coder, or a daily web user, this add-on ensures your copied data is always organized, editable, and ready to be reused instantly without ever losing focus.

<br />

## What's New
Enhanced Pinned Items: Now supports custom "Display Names" for long texts (over 50 words) and displays total character counts within the history list for better clarity.

<br />

## Hot Keys
<kbd>Windows+V</kbd>
*   **Single Press:** Opens the ClipHistory Dialog to view and manage your saved snippets.
*   **Double Press (Fast):** Instantly clears all unpinned items from your history to keep your list clean.

<br />

## Features

**1. Seamless History Management (500 Items Limit)**  
ClipHistory works silently in the background. Every time you press <kbd>Control+C</kbd>, the text is saved.  
<br />
• **How it works:** The system stores up to 500 unique items. When you reach the 501st item, the oldest unpinned entry is automatically removed to make room for the new one. This ensures your history remains current without overloading your system.

<br />

**2. Magic Auto-Paste**  
Eliminate the need for manual pasting.  
<br />
• **Step 1:** Press <kbd>Windows+V</kbd> to open the list.  
• **Step 2:** Use your arrow keys to select the snippet you need.  
• **Step 3:** Press <kbd>Enter</kbd>. ClipHistory will automatically close the dialog and paste the text directly into your active application.

<br />

**3. Advanced Pinning with Display Names**  
For long content like articles or code blocks, managing a list can be difficult. ClipHistory introduces an intelligent way to handle "Heavy" snippets.  
<br />
• **The 50-Word Trigger:** When you pin an item that contains more than 50 words, the Edit Dialog will provide a "Display Name" field.  
• **Simplifying the List:** By entering a short nickname in this field, the main history list will only show that name instead of the massive block of text.  
• **Character Awareness:** Even when using a Display Name, ClipHistory will show the total character count at the end of the item in the list, so you always know the exact size of the data you are about to paste.

<br />

**4. On-the-Fly Editing**  
Mistakes happen, but you can fix them before they are pasted.  
<br />
• **Step 1:** Locate the item in the ClipHistory list.  
• **Step 2:** Choose the Edit option. You can modify the text, update the Display Name, or toggle the "Pinned" status.  
• **Step 3:** Save the changes. The updated text is now ready for use, perfectly corrected.

<br />

**5. Structural HTML Support**  
Unlike basic managers, ClipHistory captures the formatting of web content. If you copy a link or formatted text from a website, it retains that structure, making it the perfect tool for users who work across different digital platforms.
